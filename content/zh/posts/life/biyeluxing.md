## 出发前的准备

- [x] 冲锋衣（防风，防晒，防雨，防寒）
- [x] 登山鞋或者越野鞋
- [x] 登山杖
- [x] 帽子
- [x] 防晒霜
- [x] 一次性内裤和一次性雨衣
- [x] 相机
- [x] 厚的衣服
- [x] 口罩
- [x] 学生证 身份证 驾驶证
