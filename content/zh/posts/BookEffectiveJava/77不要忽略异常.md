禁止写这样的代码：

```java
// Empty catch block ignores exception - Highly suspect!
try {
    ...
}
catch (SomeException e) { // catch异常之后什么也不做
}

```

空 catch 块违背了异常的目的，不要忽略异常。**如果你选择忽略异常，catch 块应该也至少包含一条注释，解释为什么这样做是合适的，并且应该将变量命名为 ignore：**
