**作者:** Greg Brockman (@gdb)

**互动数据:** 👍 1004 | 🔄 60 | 💬 154

### 【核心要点】
tokens 正迅速成为解决问题的通用输入方式，这标志着 AI 交互范式的根本性转变。

### 【灵感启发】
**思维模型:** 通用接口理论 - 当一种数据格式（tokens）能够承载足够丰富的语义信息时，它就能成为连接不同系统、工具、任务的通用桥梁。这类似于 HTTP 成为互联网通用协议、JSON 成为 API 通用格式。

**跨领域启发:**
- 生物学: DNA 序列作为生命信息的通用编码
- 经济学: 货币作为价值交换的通用媒介
- 物理学: 能量作为物理系统的通用度量

### 【可实践建议】
在设计 AI 应用时，优先考虑将输入/输出统一为 token 流格式，这样可以最大化与各种 LLM 和工具的兼容性，降低集成成本。
